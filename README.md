# **Boxing Mindset: Full-Stack Web Application**

<!--Elevator Pitch-->
## *💡 About the Project*
Boxing Mindset is a full-stack web application designed to serve as a "one-stop" solution for amateur boxers to manage and enhance their training. The platform includes an interval timer for structured workouts, a class attendance feature to track gym participation, and a weight management system that allows users to log weigh-ins, monitor progress, and align with boxing weight classes. The weight management feature is fully implemented with a complete frontend, backend, and database integration, while the other features are currently in development. This architecture allows the application to scale consistently, supporting future enhancements such as authentication and AI-driven training tools.
